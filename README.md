# ‚ú® Features
JSON Blend is designed to streamline your data manipulation tasks with a focus on ease of use and reliability for developers and data analysts.
- Multi-Input Merging: Easily combine data from two or more JSON inputs pasted into the interface.
- Intelligent Merging Strategy: The tool handles conflicting keys by prioritizing a deep merge approach, combining nested objects rather than simply overwriting them, ensuring minimal data loss.
- Instant Feedback: Provides immediate visual validation and helpful error messages for invalid JSON input.
- Clear Output: Merged data is automatically pretty-printed and available for immediate copying or downloading.
- Modern Stack: Built on top of React and TypeScript for a robust, performable, and maintainable application.

# ‚öôÔ∏è Local Development
If you wish to clone and run the project locally, you can set it up quickly.
## Prerequisites
You need to have Node.js (which includes npm) installed on your system.
1. Clone the Repository
``git clone [https://github.com/yaredybk/JSON-merger.git](https://github.com/yaredybk/JSON-merger.git)
cd JSON-merger``


2. Install Dependencies
Use your preferred package manager to install the project dependencies:
``# Using npm
npm install``


3. Run the Development Server
Start the application in development mode. It will be served with hot-reloading provided by Vite:
``npm run dev``


The application should now be accessible at the address displayed in your console (typically http://localhost:5173/).
4. Build for Production
To create an optimized, production-ready build for deployment:
``npm run build``


The static, optimized files will be placed in the dist/ directory.
